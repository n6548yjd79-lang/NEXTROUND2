保存したファイルには会社名やメモが含まれます。人に送らず、自分だけで保管してください。

データとプライバシー

保存場所

入力した選考情報は、あなたの端末内に保存されます。NEXTROUNDの管理者が、会社名・選考状況・締切日・メモを取得・閲覧することはありません。

端末内に保存される情報

あなたが入力した場合、会社名、選考状況、締切日、次の予定日、インターン日程、通常メモ、ログインURL・ID・パスワードが、あなたの端末内に保存されます。

これらの入力内容がNEXTROUNDの管理者に送信されたり、管理者側で閲覧されたりすることはありません。

ログイン情報

ログインURL・ID・パスワードの入力は任意です。入力した場合は、保護パスワードで暗号化して端末内に保存されます。保護パスワードを忘れた場合、保存済みログイン情報は表示できません。

バックアップ

データ保存機能で作成したファイルには、会社名・選考状況・メモなどが含まれます。人に送らず、自分だけの場所に保管してください。

GitHub Pagesでの公開

このアプリはGitHub Pagesで公開されています。ページを表示するために必要な通信情報は、GitHub側で安全管理のために扱われる場合があります。NEXTROUNDの管理者が、あなたの入力内容を取得・閲覧することはありません。

注意事項

共有端末では使わないでください。重要なパスワードは、専用のパスワード管理アプリとの併用をおすすめします。
